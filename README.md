## 📖 About

Synthlat es un bot de discord de codigo abierto en español. Que busca que cualquier persona pueda personalizar su servidor de discord que buscan algo funcional y asequible.

## 💻 Desarrollo
Se usan las siguientes teconologias:
- NextJS
- React
- Tailwind CSS
- MongoDB

### Notas Tecnicas:

El dashboard decide como se guardan los datos en la base de datos. Esta decision se tomo para facilitar el desarrollo del bot.

## 🤝 Aportes

Todos los aportes al codigo son bienvenidos. Se solicita que cada Pull Request modifique o mejore algo en especifico y no abarque demasiadas cosas al mismo tiempo para facilitar su revision.

El uso moderado de IA en las PR esta permitido. Siempre que hallas revisado personalmente el codigo, su optimizacion y escalabilidad.
